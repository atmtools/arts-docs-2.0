Workspace Variable basics_checked

OK-flag for basic variables and grids.

For example, this variable flags that the (clear-sky) atmosphere
and the surface are defined in a formally correct way, e.g. that
the sizes of atmospheric fields match the atmospheric grids.

Shall be set by basics_checkedCalc. See that WSM for treated WSVs.
