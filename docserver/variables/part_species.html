Workspace Variable part_species

part_species is an Array of Strings containing user defined input,
for particle number density calculations performed by pnd_fieldSetup.

Each single String must contain:
scattering particle type [String]:	 e.g. 'LWC', 'IWC', 'Rain'
particle size distribution [String]:	 e.g. 'MH97', 'liquid', 'H11'
sizerange-start [Numeric]:	the first particle radius(liquid)/melted radius(ice)
                            	the calculations shall start with.
sizerange-end [Numeric]:	the last particle radius(liquid)/melted radius(ice)
                          	the calculations shall end with.

NOTE: the sizerange inputs must be given in MICRONS!

The Array needs to looks like this: [''IWC-MH97-2-1000'', ''LWC-liquid-0.1-10'', ...]
