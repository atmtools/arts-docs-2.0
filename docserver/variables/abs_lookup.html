Workspace Variable abs_lookup

An absorption lookup table.

This holds an absorption lookup table, as well as all information that
is necessary to use the table to extract absorption. Extraction
routines are implemented as member functions. 

This has quite a complicated structure. See Doxygen documentation for
class GasAbsLookup for details. See also the Arts User Guide, 
Chapter 3.1 ("The gas absorption lookup table").
