Workspace Variable complex_n

Complex refractive index (n).

This matrix describes the dielectric properties of a medium. 
A typical usage of this variable is to describe the properties of
the surface (if it is assumed to be flat).

This is a two-column matrix. The first column holds the real part
of n, and the second column the imaginary part. The number of rows
should match f_grid. The case of one row is also accepted,
interpreted as that the refractive index is constant with 
frequency.

Unit:       -

Dimensions: [f_grid or 1,2]

Group: Matrix
