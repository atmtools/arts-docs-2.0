Workspace Variable abs_scalar_gas

Scalar gas absorption coefficients.

This contains the absorption coefficients for one point in the
atmosphere (one set of pressure, temperature, and VMR values). There
are two distinct cases:

Case a:    For all frequencies and all species:
Dimension: [ f_grid, abs_species ]

Case b:    For a single frequency for all species:
Dimension: [ 1,      abs_species ]

Unit: 1/m

Group: Matrix
