Workspace Method scat_data_rawCheck

Description

Method for checking the consistency of the optical properties
in the database.

This function can be used to check datafiles containing data for
randomly oriented scattering media.
It is checked whether the data is consistent. The integral over
the phase matrix should result the scattering cross section
<C_sca>.

The check is if:
<C_ext> - <C_sca> = <C_abs>

The result is printed on the screen.

Authors: Claudia Emde
