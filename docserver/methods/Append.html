Workspace Method Append

Description

Appends a workspace *in* variable to another workspace *out* variable.

This method can append a workspace variable
to another workspace variable of the same group. (E.g., a
ArrayOfMatrix to another * ArrayOfMatrix*.)

This method is not implemented for all types, just for those where an
append makes sense (see variable list below.).

As always, output comes first in the argument list!

Authors: Stefan Buehler
