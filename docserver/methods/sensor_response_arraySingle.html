Workspace Method sensor_response_arraySingle

Description

Sets up sensor_response_array from an existing sensor_response.

Fills sensor_response_array and associated variables with
corresponding non-array data. Hence, the array variables get all a
length of 1.

Authors: Patrick Eriksson
